Style Variants

Different visual styles

<!-- Default filled -->
<button class="paper-btn paper-btn-primary">Default</button>

<!-- Outline style -->
<button class="paper-btn paper-btn-outline">Outline</button>

<!-- Soft background -->
<button class="paper-btn paper-btn-soft">Soft</button>

<!-- Ghost style -->
<button class="paper-btn paper-btn-ghost">Ghost</button>

<!-- Rounded pill -->
<button class="paper-btn paper-btn-primary btn-round">Round</button>

Size Variants

Small, regular, and large

<!-- Small -->
<button class="paper-btn paper-btn-info btn-sm">Small</button>

<!-- Regular (default) -->
<button class="paper-btn paper-btn-info">Regular</button>

<!-- Large (use Tailwind utilities) -->
<button class="paper-btn paper-btn-info text-base px-6 py-3">Large</button>
