Grid System Documentation

Grid Container Classes

Class Description
grid Creates a grid container
grid-cols-{n} Sets the number of columns (1-12, or custom)
grid-rows-{n} Sets the number of rows (1-6, or custom)
grid-flow-row Items flow by row (default)
grid-flow-col Items flow by column
grid-flow-dense Dense packing algorithm

Column Span Classes

Class CSS Property Description
col-span-{n} grid-column: span n Span n columns (1-12, or full)
col-span-full grid-column: 1 / -1 Span all columns
col-start-{n} grid-column-start: n Start at column n
col-end-{n} grid-column-end: n End at column n
col-auto grid-column: auto Auto-place the item

Row Span Classes

Class Description
row-span-{n} Span n rows (1-6, or full)
row-span-full Span all rows
row-start-{n} Start at row n
row-end-{n} End at row n

Gap Classes

Class Size Description
gap-{size} 0, 1, 2, 3, 4, 5, 6, 8, 10, 12... Gap in both directions
gap-x-{size} 0, 1, 2, 3, 4, 5, 6, 8, 10, 12... Horizontal gap only
gap-y-{size} 0, 1, 2, 3, 4, 5, 6, 8, 10, 12... Vertical gap only

Gap sizes follow Tailwind's spacing scale: 1 = 0.25rem (4px), 2 = 0.5rem (8px), 4 = 1rem (16px), etc.

Bootstrap to Tailwind Migration Guide

If you're migrating from Bootstrap's grid system, here's a quick reference:

Bootstrap Class Tailwind Equivalent
.container container mx-auto px-4
.row grid grid-cols-12 gap-4
.col-6 col-span-6
.col-md-4 col-span-12 md:col-span-4
.col-lg-3 col-span-12 lg:col-span-3
.offset-3 col-start-4
.g-4 (gutters) gap-4
