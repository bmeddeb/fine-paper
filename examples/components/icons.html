Icon Libraries

Nucleo for core UI icons, with Font Awesome for supplemental symbols.

Nucleo Quick Start

How to use Nucleo icons in your project

1. Add the icon class

Use nc-icon + icon name

2. Style with Tailwind

Add color and size classes

3. Done!

Icons inherit font styles

<!-- Basic usage -->
<i class="nc-icon nc-bell-55"></i>

<!-- With Tailwind color -->
<i class="nc-icon nc-bell-55 text-[var(--paper-primary)]"></i>

<!-- With size -->
<i class="nc-icon nc-bell-55 text-2xl"></i>

<!-- In a button -->
<button class="paper-btn paper-btn-primary">
  <i class="nc-icon nc-bell-55 mr-2"></i> Notifications
</button>

Font Awesome Quick Start

Supplemental icons for missing symbols

1. Add the base class

Use fa + fa-[name]

2. Style with Tailwind

Add size and color utilities

3. Use alongside Nucleo

Great for gaps in the set

<!-- Basic usage -->
<i class="fa fa-star"></i>

<!-- With Tailwind color -->
<i class="fa fa-star text-[var(--paper-primary)]"></i>

<!-- With size -->
<i class="fa fa-star text-2xl"></i>

<!-- In a button -->
<button class="paper-btn paper-btn-primary">
  <i class="fa fa-star mr-2"></i> Favorite
</button>
