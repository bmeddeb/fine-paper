Collapsible Panels

Accordion-style expandable cards with Alpine.js

What is Paper Dashboard?

Paper Dashboard is a beautiful Bootstrap 4 admin dashboard template ported to Tailwind CSS 4.1 with Alpine.js for interactivity.

How do I customize the theme?

Customize the CSS variables in the app.css file to change colors, fonts, and spacing throughout the entire dashboard.

Is it responsive?

Yes! All components are fully responsive and work great on mobile, tablet, and desktop devices.

<div class="paper-card" x-data="{ open: false }">
  <div class="paper-card-header flex items-center justify-between cursor-pointer" @click="open = !open">
    <h4 class="font-semibold">Panel Title</h4>
    <i class="nc-icon nc-minimal-down transition-transform" :class="open ? 'rotate-180' : ''"></i>
  </div>
  <div class="paper-card-body" x-show="open" x-collapse>
    <p>Collapsible content here...</p>
  </div>
</div>

Task Lists

Interactive todo/task lists

Today's Tasks

<div class="paper-card" x-data="{ tasks: [
  { text: 'Review PRs', done: true },
  { text: 'Update docs', done: false }
] }">
  <div class="paper-card-header">
    <h4>Today's Tasks</h4>
    <span x-text="tasks.filter(t => t.done).length + '/' + tasks.length"></span>
  </div>
  <div class="divide-y">
    <template x-for="task in tasks">
      <label class="px-4 py-3 flex items-center gap-3">
        <input type="checkbox" x-model="task.done">
        <span :class="task.done ? 'line-through' : ''" x-text="task.text"></span>
      </label>
    </template>
  </div>
</div>

Panel Classes Reference

Card Structure Classes

Class Description
paper-card Base card container with rounded corners and shadow
paper-card-header Card header section with bottom border
paper-card-body Card body section with padding
paper-card-footer Card footer section with top border
paper-card-title Styled title text for card headers

Useful Utility Classes

Class Use Case
overflow-hidden For cards with colored headers
divide-y Add borders between list items
text-center Center-aligned content cards
cursor-pointer Clickable/collapsible headers
x-collapse Alpine.js smooth collapse animation
