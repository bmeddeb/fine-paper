Live Email Validation

Real-time validation with Alpine.js

We'll never share your email. Valid email address! Please enter a valid email address. 

<div x-data="{
  email: '',
  touched: false,
  get isValid() {
    return /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(this.email);
  },
  get state() {
    if (!this.touched || !this.email) return 'neutral';
    return this.isValid ? 'success' : 'error';
  }
}">
  <input
    type="email"
    class="paper-input"
    :class="{
      'input-success': state === 'success',
      'input-error': state === 'error'
    }"
    x-model="email"
    @blur="touched = true"
  >
  <p :class="state === 'error' ? 'text-red-500' : 'text-gray-500'">
    <span x-show="state === 'success'">Valid!</span>
    <span x-show="state === 'error'">Invalid email.</span>
  </p>
</div>

Password Strength

Visual strength indicator

  • At least 8 characters
  • Upper and lowercase letters
  • At least one number
  • At least one special character 
<div x-data="{
  password: '',
  get strength() {
    let score = 0;
    if (this.password.length >= 8) score++;
    if (this.password.length >= 12) score++;
    if (/[a-z]/.test(this.password) && /[A-Z]/.test(this.password)) score++;
    if (/\d/.test(this.password)) score++;
    if (/[^a-zA-Z0-9]/.test(this.password)) score++;
    return score;
  }
}">
  <input type="password" x-model="password">
  <div class="flex gap-1">
    <template x-for="i in 5">
      <div class="h-1.5 flex-1 rounded-full"
        :class="i <= strength ? 'bg-emerald-500' : 'bg-gray-200'"></div>
    </template>
  </div>
</div>

Required Fields

Mark and validate required inputs

<!-- Required Field Label -->
<label class="paper-label">
  Full Name <span class="text-[var(--paper-danger)]">*</span>
</label>

<!-- Validation Logic -->
<div x-data="{
  name: '',
  touched: false,
  get error() {
    if (!this.touched) return '';
    if (!this.name.trim()) return 'Name is required.';
    return '';
  }
}">
  <input
    type="text"
    :class="error ? 'input-error' : ''"
    x-model="name"
    @blur="touched = true"
  >
  <p x-show="error" x-text="error" class="text-xs text-red-500"></p>
</div>

Pattern Validation

Regex patterns for common inputs

Format: +1 (555) 000-0000 Invalid phone number format. Valid phone number!

Must start with http:// or https:// Invalid URL format. Valid URL!

16 digits, spaces optional. Invalid card number. Valid card format!

5 digits, or ZIP+4 format. Invalid ZIP code. Valid ZIP code! 

<!-- Pattern Validation Example -->
<div x-data="{
  value: '',
  touched: false,
  get valid() { return /^\d{5}(-\d{4})?$/.test(this.value); }
}">
  <input
    type="text"
    :class="touched && value && !valid ? 'input-error'
           : touched && valid ? 'input-success' : ''"
    x-model="value"
    @blur="touched = true"
  >
</div>
