Why single-column still wins on mobile.

Single-column layouts reduce cognitive load. With 57% of viewing time above the fold and 74% in the first two screens, the goal is to make the first paragraphs feel complete while inviting the scroll. This edition keeps the key message in the opening lines, then layers supporting context below.

Research

Reflow at 320px is non-negotiable.

WCAG 1.4.10 requires single-column content to function at 320 CSS pixels. This feed collapses side rails, widens line-height, and keeps tap targets above 24x24px so the layout remains usable at high zoom.

Use CSS Grid for the overall column and Flexbox inside each card to keep metadata aligned without resorting to visual reordering.

Practice

Line length is a layout decision.

The paper recommends 50-75 characters per line (about 600-700px wide) for comfortable reading. Single-column layouts make this easy: set a max-width, keep text left aligned, and avoid justified paragraphs that create rivers.

The feed stays centered so the reading column feels consistent from hero to footer.

Performance

Reserve space to protect CLS.

Always specify media dimensions or aspect ratios before images load. The target CLS score is 0.1 or lower. When content is long, use content-visibility to skip off-screen rendering without breaking search and focus.

The card system here keeps image frames stable and metadata consistent, even when text wraps.

Reading comfort

Lines should average 80 characters or less. We cap the content column to maintain readability and keep headings short enough to fit above the fold.

Accessibility and reflow

Single-column layouts are the easiest way to meet reflow requirements. Keep DOM order aligned to visual order, maintain landmarks, and ensure touch targets meet the 24x24px minimum.

Reflow at 320 CSS pixels without horizontal scrolling.

Keep focus order aligned to the reading order.

Provide a skip link and landmark regions for fast navigation.

Performance notes

Containment and content-visibility shrink the rendering cost of long feeds while keeping text searchable. Combine these with aspect ratios to keep the layout stable.

.section {
  content-visibility: auto;
  contain-intrinsic-size: auto 500px;
}

.media {
  aspect-ratio: 16 / 9;
  object-fit: cover;
}

Target CLS <= 0.1 by reserving space before images load.

Load more

NNGroup and Baymard both note that explicit load controls outperform endless scrolling for long reading lists. This keeps orientation and avoids the illusion of completeness.
