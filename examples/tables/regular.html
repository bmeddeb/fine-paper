Quick Reference

Table classes and modifiers

Class Description
.paper-table Base table styling with border-bottom dividers
.paper-table-striped Alternating stripe backgrounds
.paper-table-bordered Full borders on all table cells
.paper-table-hover Highlight entries on hover
.paper-table-sm Compact cell padding (0.5rem)
.paper-table-lg Relaxed cell padding (1rem)
.text-start Left-align column content
.text-center Center-align column content
.text-end Right-align column content
.overflow-x-auto Enable horizontal scroll (wrapper)
